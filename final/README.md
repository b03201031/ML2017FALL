## Requirements

##### Execution Environment

- python 3

##### Packages required

- [gensim]  (3.1.0)
- [numpy]  (1.14.0)
- [pandas]  (0.22.0)

## Setup
- command to run prediction

    put predict.sh and predict.py in the same folder and type bash predict.sh PREDICTION_PATH in command line
    eg. bash predict.sh pre.csv
